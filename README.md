# Ella Fellas

Unofficial fan-run news and tour information site for country artist Ella Langley.

**Not affiliated with Ella Langley, her management, label, or official merchandise.**

## Stack

- Next.js 15 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3
- Supabase (Postgres) for newsletter signup + alerts
- next-mdx-remote for content
- Deployed on Vercel

## Develop

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build && pnpm start
```
